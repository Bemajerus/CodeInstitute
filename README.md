## Target Audience

The target audience for this website is not only Vasco da Gama fans who want to learn more about the club's history but also anyone interested in understanding the history of the fight against racism in early 1900s Brazil, where Vasco was a pioneer. The website contains pages on the history, the struggle against racism, and legendary players of the club. It aims to provide valuable insights and information for a diverse range of visitors interested in these topics.
